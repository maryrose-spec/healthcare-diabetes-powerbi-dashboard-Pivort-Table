# Sales Performance Analysis Using Excel 

## Project Overview

This project analyzes sales performance data using Pivort Table to build an interactive dashboard for visualization and reporting, starting with Data Cleaning in Excel.

The goal of this project is to identify key sales trends, top-performing categories, regions, employees, and the impact of discounts on revenue generation.

---

## Tools Used
- Microsoft Excel
- Pivort Table

## Dataset Information

The dataset contains sales transaction records with the following fields:
Date	
Product
Category	
Sales Rep	
City	
Units	
Price (₦)	
Total Sales (₦)

---

## Business Questions Answered

### Sales Performance Analysis

1. What is the total Sale by City?
2. Highest sold Product?
3. Which employee handled the most orders?
4. Which Category generated the highest Amount?
5. What are the monthly sales trends?

## Skills Demonstrated

1. Data Cleaning 
2. Dashbord Creation using Pivort Table 
3. CSV files were imported into Microsoft Excel.
4. Pivot Tables were created to summarize key metrics.
5. Charts and visualizations were developed.
6. An interactive dashboard was designed to present business insights.
