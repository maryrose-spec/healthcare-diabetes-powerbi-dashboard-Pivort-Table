# Healthcare Diabetes Analysis Dashboard

## Project Overview

This project explores a healthcare diabetes dataset using Power BI to identify patterns, trends, and risk factors associated with diabetes.

The dashboard provides interactive insights into patient demographics, glucose levels, BMI, blood pressure, and diabetes outcomes.

## Objectives

* Analyze diabetes prevalence among patients
* Identify relationships between BMI and diabetes risk
* Explore the impact of glucose levels on diabetes outcomes
* Investigate diabetes distribution across age groups
* Build an interactive healthcare analytics dashboard

## Dataset Features

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
* Outcome

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Key Insights

* Obese patients showed a higher prevalence of diabetes.
* Elevated glucose levels were associated with diabetic outcomes.
* Diabetes risk varied across age categories.
* BMI and glucose emerged as important indicators of diabetes risk.

## Dashboard Features

* KPI Cards
* Interactive Slicers
* Donut Charts
* Trend Analysis
* Risk Segmentation
* Comparative Visualizations
